# NordicMdkTest
用于试验Noridic52832控制外设的试验台工程，其他正式工程的小帮手。
工程说明：
0：本试验台使用的平台信息为：
	MCU:		Nordic52832
	编译器：	MDK5
1：建立了一个MDK的初步工程，还未有初步的功能。
2：基本工程已经可以运行，gpio的led测试已经可以运行。
3：初步测试led功能。
	初步测试key功能。
4：初步测试uart功能。
5：初步加入iic功能，硬件方式。
	由于暂时没有iic外设，所以没法用实际通信的方式测试是否正确，只是mcu单方面发送数据，从波形来看一下第一个字节的波形，初步看应该是正确的。
6：使用MPU6050模块，初步测试可以获取Accel和Gyro的原始值。
